Регламент

Блиц-турнир. Проводим с 12-го декабря по 26-декабря включительно.
Участвует 8 человек. Если кто-то пожелает присоединиться после начала турнира, то принимаем.
Играем по круговой схеме с контролем времени 5 минут + 3 секунды добавление на ход.
Каждый играет с соперником две партии, одну белыми, одну чёрными.
Конкретные дата и время игр не фиксированы, играем по готовности, но не затягиваем.
Тай-брейк коэффициенты: количество побед, Neustadtl (Sonneborn-Berger), Koya.

По окончании круговой части турнира играем тай-брейк между первыми четырьмя местами.
Жеребьёвку проведём выбором из корзины вслепую. В полуфинале играются две игры с контролем 5+3. В случае ничьей играется армагеддон: одна партия, белые 5+3, чёрные 4+3, цвет определяем подбрасыванием монеты, ничья трактуется в пользу чёрных. За третье место играют проигравшие в полуфинале по тому же регламенту. В финале играем 2 игры с контролем 5+3, при ничьей ещё две партии с тем же контролем, дальше армагеддон.
